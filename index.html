¿Qué son los frameworks?

Es una estructura de trabajo en la programación para desarrollar apps, webs de forma más eficiente y rápida.

¿Que es Bootstrap CSS?

Es un framework que ayuda a que el desarrollo web sea más fluido, ya que proporciona plantillas que cualquiera puede usar y de manera grtatuita.

Maneras de agregar Bootstrap a un proyecto.

La forma más rápida sería mediante enlaces CDN, que se puedes copiar en la página principal de Bootstrap.

Otra forma sería descargando los archivos de Bootstrap e incluirlos en una carpeta y cargarlos en un archivo HTML.

¿Qué son los breackpoints de bootstrap?

Son medidas para controlar cuando queremos adaptar nuestra maquetacion a diversos tamaños.

Utilities de Bootstrap

Se generan con una API y se usa para modificar un predeterminado conjunto de clases usando SASS.

Para usarlos primero se puede descargar la biblioteca de Bootstrap o usar un CDN, como ya se mencionó.

Apartir de ello se puedes utilizar las clases de utilities directamente en el HTML.
